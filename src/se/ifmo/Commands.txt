help - справка
info - вывод информации о коллекции
show - вывод элементов коллекции в строковом формате
insert key - добавить элемент с заданным ключём
update id - обновить элемент с заданым ключём
remove_key key - удалить элемент по ключу
clear - очистить коллекцию
save - сохранить коллекцию
execute_script file_name - считать из файла скрипт и исполнить
exit - завершение работы программы без сохранения в файл
remove_lower {element} - удалить из коллекции все элементы меньше чем заданный
replace_if_lower key {element} - заменить значение по ключу, если новое больше старого
remove_greater_key key - удалить все элементы, ключ которых превышает заданный
filter_by_venue venue - вывод всеъ элементов с заданным значением venue
print_descending - вывести элементы коллекции в порядке убывания
print_field_descending_type type - вывести значения поля type в порядке убывания
